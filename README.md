# Tool2-civ442-Turf
My Repository for tool 2 This is an air pollution model for a hypothetical flat region where an Ai data center is proposed to be built. To account for the data center a power plant is to be built to accommodate its power requirements. The data center is deciding which power plant to build A coal fired power plant or a natural gas powerplant. Each power plant will be evaluated on the escaping particulate matter (concentration), cost, and power demands for the data center. The emission rate will be calculated depending on power requirements of the data center. The user will be able to adjust the power requirements for the data center. Ambient values of pm, power generations, and power requirements will be based on literature and based on their respective efficiencies.
