# Tool2-civ442-Turf
My Repository for tool 2 
This is a dispersion airpolution model for a hypothetical flat region with a town where an Ai data center is proposed to be built. 
To account for the data center a power plant is to be built to accomidate its power requirements. 
The data center is desciding which power plant to build A coal fired power plant, a coal powerplant with a baghouse, or a natrual gas powerplant.
Each power plant will be evaluated on the despersion of particulate matter ( range and concentration), cost, and power demands for the data center.
The dispersion will be modled via gaussian plume model where wind, atmospheric stability, emission rate and stack height will be taken into account 
The emission rate will be calculated depending on power requirements of the data center and the town. 
The user will be able to adjust the power requirements for the data center, wind speed and direction, stack height, and atmospheric stability. 
Ambient values of pm, power generations, and power requirments will be based on literature and based on their respective effeceiecies.
